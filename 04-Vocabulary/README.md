## 💡 Learning Tips

### 1. **Spaced Repetition**
Review words at increasing intervals:
- Day 1: Learn new words
- Day 2: Review
- Day 4: Review
- Day 7: Review
- Day 14: Review

### 2. **Context Learning**
- Always learn words with example sentences
- Create your own sentences using new vocabulary
- Connect words to real-life situations

### 3. **Active Recall**
- Cover the Marathi column and try to recall the word
- Cover the English column and translate back
- Practice speaking the words aloud

### 4. **Thematic Grouping**
- Learn related words together (e.g., all family terms)
- Create mental associations between related concepts
- Use mind maps to visualize connections

### 5. **Daily Practice Routine**
```
Morning: Review 10 words from frequency list
Afternoon: Practice themed vocabulary
Evening: Write sentences with new words
```

## 🔤 How to Use Vocabulary Tables

Each vocabulary file uses this format:

| English | Marathi | Transliteration | Example Sentence |
|---------|---------|-----------------|------------------|
| Hello | नमस्कार | namaskār | नमस्कार, तुम्ही कसे आहात? |

**Column Guide:**
- **English**: The English translation
- **Marathi**: The word in Devanagari script
- **Transliteration**: Romanized pronunciation guide
- **Example Sentence**: Usage in context (Marathi + English)

## 🗣️ Pronunciation Tips

### Vowel Sounds
| Symbol | Sound | Example |
|--------|-------|---------|
| ā | "a" in father | आई (āī) - mother |
| ī | "ee" in see | दीध (dīdh) - milk |
| ū | "oo" in boot | सूर (sūr) - tune |
| e | "e" in they | एक (ek) - one |
| o | "o" in go | ओला (olā) - wet |

### Consonant Clusters
- त्र (tr) - in त्री (trī)
- ज्ञ (jñ) - in ज्ञान (jñān)
- श्र (śr) - in श्री (śrī)

## 📝 Practice Exercises

After studying each vocabulary file:

1. **Write** - Copy each word 3 times in Devanagari
2. **Speak** - Say the word aloud with correct pronunciation
3. **Use** - Create 2-3 original sentences
4. **Review** - Test yourself without looking at answers

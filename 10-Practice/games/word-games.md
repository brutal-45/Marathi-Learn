# Word Games for Marathi Learning

Fun and interactive word games to practice your Marathi vocabulary and spelling.

---

## Game 1: Word Search (शब्द शोध)

Find the hidden Marathi words in the grid below.

### Puzzle 1: Animals (प्राणी)

Find these animal names in the grid:
- कुत्रा (dog)
- मांजर (cat)
- गाय (cow)
- वाघ (tiger)
- हत्ती (elephant)

```
कु त र ा म ा ं ज र ह
त ह त त ी व ा घ ग ा
म ा ं ज र क ग ा य य
व ा घ ह त त ी क त क
म ा क ड म ा ं ज र र
```

### Answers 1:
- कुत्रा: Row 1, left to right
- मांजर: Row 3, left to right (and Row 5)
- गाय: Row 3, near end
- वाघ: Row 2 and Row 4
- हत्ती: Row 2 and Row 4

## Game 2: Crossword Puzzle (वर्गपेच)

### Puzzle: Basic Vocabulary

**Across (आडवे):**
1. I (pronoun) - म \_\_\_
3. Water - प \_\_\_ नी
5. Mother - \_\_\_ ई
7. One - \_\_\_ क

**Down (उभे):**
2. He - \_\_\_ 
4. Rice - भ \_\_\_ त
6. Two - द \_\_\_ 
8. Tea - च \_\_\_ ा

### Crossword Grid:

```
    1   2   3   4
  ┌───┬───┬───┬───┐
1 │ म │   │ प │   │
  ├───┼───┼───┼───┤
2 │ ा │ त │ ा │ भ │
  ├───┼───┼───┼───┤
3 │ झ │ ा │ ण │ ा │
  ├───┼───┼───┼───┤
4 │ ा │ त │ ी │ त │
  └───┴───┴───┴───┘
```

### Answers 2:
**Across:**
1. मी (mī) - I
3. पाणी (pāṇī) - Water
5. आई (āī) - Mother
7. एक (ek) - One

**Down:**
2. तो (to) - He
4. भात (bhāt) - Rice
6. दोन (don) - Two
8. चहा (cahā) - Tea

## Game 3: Word Scramble (अक्षरे जुळवा)

Unscramble the letters to form correct Marathi words.

1. ा म ज ा ठ म र → \_\_\_\_\_\_\_\_
   
   **Answer:** मराठी (Marathi)

2. न व क ा य → \_\_\_\_\_\_
   
   **Answer:** नाव काय (name what)

3. भ ा ज क र ी → \_\_\_\_\_\_\_\_
   
   **Answer:** भाकरी (bhakri)

4. व स र ा ा म → \_\_\_\_\_\_\_\_
   
   **Answer:** स्वारस (interesting) OR रविवार (Sunday)

5. ा र ल प ढ ं व प → \_\_\_\_\_\_\_\_\_\_
   
   **Answer:** पांढरा (white)

## Game 5: Word Chain (शब्द साखळी)

Start with a word, and the next word must begin with the last letter of the previous word.

### Example Chain:
मराठी → ईद → दिवस → सकाळ → ळी (not possible) → Let's try again!

### Play:
1. Start with: **नमस्कार** (namaskār)
2. Next word must start with **र**: \_\_\_\_\_\_\_\_
3. Next word must start with that word's last letter: \_\_\_\_\_\_\_\_
4. Continue for 5 words!

### Sample Answer:
1. नमस्कार (namaskār)
2. रविवार (ravivār) - Sunday
3. रस्ता (rastā) - Road
4. आई (āī) - Mother
5. ईद (īd) - Festival
6. दिवस (divas) - Day

## Game 6: Fill in the Vowels (स्वर भरा)

Complete the words by adding the correct vowel signs (मात्रा).

1. क \_\_ र → कार (car) - Add \_\_\_
   
   **Answer:** Add ा (aa) → कार

2. ग \_\_ द \_\_ → गुड़े (sweets) - Add \_\_\_ and \_\_\_
   
   **Answer:** Add ु (u) and े (e) → गुडे

3. श \_\_ ळ \_\_ → शाळा (school) - Add \_\_\_ and \_\_\_
   
   **Answer:** Add ा (aa) and ा (aa) → शाळा

4. म \_\_ र \_\_ ठ \_\_ → मराठी (Marathi) - Add \_\_\_, \_\_\_, \_\_\_
   
   **Answer:** Add ा (aa), nothing on ठ, ी (ī) → मराठी

5. भ \_\_ ज \_\_ → भाजी (vegetable) - Add \_\_\_ and \_\_\_
   
   **Answer:** Add ा (aa) and ी (ī) → भाजी

## Game 7: Odd One Out (वेगळा शब्द ओळखा)

Find the word that doesn't belong in each group.

### Group 1:
- आई (mother)
- वडील (father)
- **पाणी (water)**
- भाऊ (brother)

**Answer:** पाणी - The others are family members

### Group 2:
- सोमवार (Monday)
- मंगळवार (Tuesday)
- **जानेवारी (January)**
- बुधवार (Wednesday)

**Answer:** जानेवारी - The others are days of the week

### Group 3:
- लाल (red)
- निळा (blue)
- **फूल (flower)**
- पिवळा (yellow)

**Answer:** फूल - The others are colors

### Group 4:
- कुत्रा (dog)
- मांजर (cat)
- गाय (cow)
- **भात (rice)**

**Answer:** भात - The others are animals

### Group 5:
- एक (one)
- दहा (ten)
- **नाव (name)**
- पाच (five)

**Answer:** नाव - The others are numbers

## Game 8: Rhyming Words (आडवी जोडी)

Find Marathi words that rhyme with the given word.

1. **आई (mother)** - Find words ending in ई:
   - बहीण (sister) - No ❌
   - **माझी (my)** - Yes ✓
   - **तुझी (your)** - Yes ✓
   - **लाडी (sweets/doll)** - Yes ✓

2. **घर (house)** - Find words ending in अर:
   - **बाजार (market)** - Yes ✓
   - **नगर (city)** - Yes ✓
   - **पाणी (water)** - No ❌

3. **पुस्तक (book)** - Find words ending in अक:
   - **पथक (group)** - Yes ✓
   - **रथक (chariot wheel)** - Yes ✓
   - **मोर (peacock)** - No ❌

## Game 9: Picture-Word Association (चित्र-शब्द जोडी)

Match the description with the correct Marathi word.

1. I am green and grow in the ground:
   - a) फूल (flower)
   - b) **गवत (grass)**
   - c) झाड (tree)
   
   **Answer:** गवत (grass)

2. I have four legs and say "म्याँव":
   - a) कुत्रा (dog)
   - b) गाय (cow)
   - c) **मांजर (cat)**
   
   **Answer:** मांजर (cat)

3. I come after Sunday:
   - a) शनिवार (Saturday)
   - b) **सोमवार (Monday)**
   - c) रविवार (Sunday)
   
   **Answer:** सोमवार (Monday)

4. I am white and we drink me:
   - a) पाणी (water)
   - b) **दूध (milk)**
   - c) चहा (tea)
   
   **Answer:** दूध (milk)

## Game 10: Sentence Builder (वाक्य बनवा)

Arrange the words to form correct sentences.

### Exercise 1:
Words: आहे / नमस्कार / मी / माझे / राम / नाव

**Correct Order:** नमस्कार, माझे नाव राम आहे.
**Translation:** Hello, my name is Ram.

### Exercise 2:
Words: जातो / शाळेत / मी / रोज

**Correct Order:** मी रोज शाळेत जातो.
**Translation:** I go to school every day.

### Exercise 3:
Words: आवडते / मला / मराठी / भाषा

**Correct Order:** मला मराठी भाषा आवडते.
**Translation:** I like the Marathi language.

### Exercise 4:
Words: खेळत / मुले / बागेत / आहेत

**Correct Order:** मुले बागेत खेळत आहेत.
**Translation:** Children are playing in the garden.

### Exercise 5:
Words: काय / तुमचे / आहे / नाव

**Correct Order:** तुमचे नाव काय आहे?
**Translation:** What is your name?

## Game 11: Complete the Proverb (म्हण पूर्ण करा)

Complete these common Marathi sayings.

1. अति ते \_\_\_\_\_\_\_\_ (Excess is bad)
   
   **Answer:** अति ते **बेगडी** OR the more common: अति सर्वत्र वर्ज्य

2. प्रयत्नांती \_\_\_\_\_\_\_\_ (After effort comes success)
   
   **Answer:** प्रयत्नांती **परमेश्वर**

3. जसे दृष्टी तसे \_\_\_\_\_\_\_\_ (As the vision, so the world)
   
   **Answer:** जसे दृष्टी तसे **सृष्टी**

4. ज्ञान हेच \_\_\_\_\_\_\_\_ (Knowledge is true wealth)
   
   **Answer:** ज्ञान हेच **खरे धन**

## Game 12: Rapid Fire (झपाट्याने उत्तर द्या)

Answer as many as you can in 60 seconds!

1. What is 'sun' in Marathi? → **सूर्य** (sūrya)
2. What is 'moon' in Marathi? → **चंद्र** (candra)
3. Count 1-5 in Marathi → **एक, दोन, तीन, चार, पाच**
4. Say 'good night' in Marathi → **शुभ रात्री**
5. What is 'book' in Marathi? → **पुस्तक**
6. How do you say 'I love Marathi'? → **मला मराठी आवडते**
7. What color is 'पिवळा'? → **Yellow**
8. How many days in a week? → **सात** (seven)
9. What is 'school' in Marathi? → **शाळा**
10. Say 'thank you' in Marathi → **धन्यवाद**

## Tips for Playing

1. **Start Easy**: Begin with matching and odd-one-out games
2. **Practice Daily**: Spend 10-15 minutes on word games
3. **Play with Friends**: Challenge each other
4. **Create Your Own**: Make new puzzles using vocabulary you've learned
5. **Have Fun**: Learning is more effective when enjoyable!

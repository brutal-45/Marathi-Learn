## Answer Key

### Quiz 1: Vowels
1. **A** - अ is the short 'a' sound (schwa)
2. **B** - ई is the long 'ī' sound
3. **C** - ए is the 'e' sound as in 'hey'
4. **A** - ऊ represents the long 'ū' sound
5. **B** - ऐ is the 'ai' diphthong

### Quiz 2: Consonants
6. **B** - क represents 'ka'
7. **B** - म represents 'ma'
8. **C** - च is 'cha'
9. **B** - त is dental (tongue touches teeth), ट is retroflex (tongue curls back)
10. **D** - य represents 'ya'

### Quiz 3: Barakhadi
11. **B** - क + इ = की (keemat allows ki/kī based on context)
12. **B** - ग + ऊ = गू
13. **B** - चे = च + ए
14. **A** - प + ओ = पो
15. **B** - म + औ = मौ

### Quiz 4: Conjuncts
16. **D** - क्ष represents क् + ष (k + ṣa)
17. **A** - त्र = त + र combined
18. **A** - ज + ञ = ज्ञ (as in विद्या - knowledge)
19. **B** - श्र = श + र (as in श्री)
20. **A** - क्त is formed by क + त

### Quiz 5: Words
21. **A** - म is the first letter of मराठी
22. **B** - 3 consonants: भ, र, त (with inherent vowels)
23. **C** - नमस्कार has ा (aa matra) on स and implicit a's
24. **C** - Both describe श + ा + ळ + ा = शाळा
25. **B** - मुंबई has 4 components: मु + ं + ब + ई (or 3 letters म, ब, ई with matras)

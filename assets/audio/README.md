# Audio Assets

This directory contains audio files for the Learn Marathi project.

## Purpose

Store audio learning materials including:

- Alphabet pronunciation recordings
- Vocabulary word pronunciations
- Conversation dialogues
- Listening practice exercises 
- Native speaker recordings

## Guidelines

- Use clear, high-quality recordings
- Keep files reasonably sized for web delivery
- Use descriptive filenames (e.g., `vocabulary-family-mother.mp3`)
- Include speaker information when relevant
- Organize by category if collection grows

## Supported Formats

- MP3 (recommended for compatibility)
- WAV (for high-quality recordings)
- OGG (for web optimization)
- M4A (for Apple devices)

## Recording Tips

1. Use a quiet environment
2. Speak clearly at natural speed
3. Include both male and female voices when possible
4. Provide both slow and normal speed versions for beginners

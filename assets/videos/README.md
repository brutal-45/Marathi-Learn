# Video Assets

This directory contains video content for the Learn Marathi project.

## Purpose

Store video learning materials including:

- Alphabet writing demonstrations
- Pronunciation tutorials
- Conversation practice videos
- Cultural documentaries
- Grammar explanations
- Interactive lessons

## Guidelines

- Keep videos concise and focused
- Include clear audio and visuals
- Add subtitles when possible
- Use descriptive filenames (e.g., `lesson-01-greetings.mp4`)
- Consider multiple quality options for different bandwidths

## Supported Formats

- MP4 (recommended for compatibility)
- WebM (for web optimization)
- MOV (for source files)

## Video Categories

| Category | Description |
|----------|-------------|
| Lessons | Structured learning videos |
| Pronunciation | Speaking and listening practice |
| Culture | Cultural content and documentaries |
| Writing | Handwriting and script tutorials |
| Conversations | Real-world dialogue examples |

## Production Tips

1. Use good lighting and clear audio
2. Keep videos under 10 minutes when possible
3. Include chapter markers for longer videos
4. Provide transcripts for accessibility
5. Consider both formal and informal language variants

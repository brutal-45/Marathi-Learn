# Number Translation Practice | संख्या अनुवाद सराव

## Introduction | परिचय

This exercise focuses on translating numbers between English and Marathi, helping you build fluency in both numeral systems and the Indian numbering terminology.

## Practice Tips | सराव टिप्स

1. **Practice daily** - Translate 5-10 numbers every day
2. **Use flashcards** - Create cards with English on one side, Marathi on the other
3. **Think in Marathi** - When you see numbers, try to say them in Marathi
4. **Write it down** - Writing helps reinforce memory
5. **Use in real life** - Practice with prices, dates, phone numbers

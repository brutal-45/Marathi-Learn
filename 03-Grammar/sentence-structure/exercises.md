# Sentence Structure Exercises | वाक्य रचना अभ्यास

## Overview | सारांश

This comprehensive practice section consolidates your understanding of Marathi sentence structure, including word order, questions, and negation. Work through these exercises to build fluency and confidence.

हा सर्वसमावेशक सराव विभाग शब्दक्रम, प्रश्न आणि नकार यांच्यासह मराठी वाक्य रचनेची तुमची समज दृढ करतो. प्रवाह आणि आत्मविश्वास वाढवण्यासाठी या अभ्यासांमधून जा.

## Exercise Set 1: Word Order | अभ्यास संच १: शब्दक्रम

### 1.1 Rearrange Words | शब्द पुनर्क्रमित करा

Arrange the words in correct SOV order:

शब्द योग्य SOV क्रमात लावा:

| Words | Your Answer | Correct Answer |
|-------|-------------|----------------|
| खातो / मी / भात | | मी भात खातो |
| पिते / पाणी / ती | | ती पाणी पिते |
| वाचतो / पुस्तक / तो | | तो पुस्तक वाचतो |
| बोलतात / मराठी / ते | | ते मराठी बोलतात |
| जाते / शाळेला / ती | | ती शाळेला जाते |
| करतो / काम / तो | | तो काम करतो |
| खेळतात / मुले / क्रिकेट | | मुले क्रिकेट खेळतात |

## Exercise Set 3: Negation | अभ्यास संच ३: नकार

### 3.1 Convert to Negative | नकारात्मक बनवा

Convert affirmative sentences to negative:

होकारात्मक वाक्ये नकारात्मक बनवा:

| Affirmative | Your Answer | Correct Answer |
|-------------|-------------|----------------|
| मी जातो | | मी जात नाही |
| ती येते | | ती येत नाही |
| तो खातो | | तो खात नाही |
| ते बोलतात | | ते बोलत नाहीत |
| आपण शिकतो | | आपण शिकत नाही |
| राम वाचतो | | राम वाचत नाही |
| सीता गाते | | सीता गात नाही |
| मुले खेळतात | | मुले खेळत नाहीत |

### 3.3 Future Tense Negation | भविष्यकाळ नकार

# Conjunctions in Marathi | मराठी उभयान्वयी अव्यय

## Introduction | परिचय

Conjunctions (उभयान्वयी अव्यय - *ubhayānvayī avyay*) are words that connect words, phrases, or clauses. They help create complex sentences and show relationships between ideas.

## Types of Conjunctions | उभयान्वयी अव्ययांचे प्रकार

### By Function | कार्यानुसार

| Type | Marathi Name | Function |
|------|--------------|----------|
| Coordinating | समानांतर | Join equal elements |
| Subordinating | अधीन | Join dependent clauses |
| Correlative | परस्पर | Work in pairs |

### Common Categories | सामान्य श्रेणी

| Category | Function | Examples |
|----------|----------|----------|
| Addition | Add information | आणि (and) |
| Contrast | Show opposition | पण (but) |
| Choice | Present alternatives | किंवा (or) |
| Cause | Show reason | कारण (because) |
| Result | Show effect | म्हणून (so/therefore) |
| Condition | Set conditions | तर (if/then) |

## Most Common Conjunctions | सर्वात सामान्य उभयान्वयी अव्यय

| English | Marathi | Transliteration | Usage |
|---------|---------|-----------------|-------|
| and | आणि | āṇi | Addition |
| but | पण | paṇ | Contrast |
| or | किंवा | kivā | Choice |
| if/then | तर | tar | Condition |
| so/therefore | म्हणून | mhaṇūn | Result |
| because | कारण | kāraṇ | Cause |

## Basic Sentence Patterns | मूळ वाक्य प्रकार

### Connecting Words
```
राम आणि श्याम = Ram and Shyam
दोन किंवा तीन = Two or three
```

### Connecting Phrases
```
घरी जा आणि जेव = Go home and eat
इथे राहा किंवा तिथे जा = Stay here or go there
```

### Connecting Clauses
```
तो आला पण बोलला नाही = He came but didn't speak
पाऊस पडेल तर येऊ नका = If it rains, don't come
```

## Key Characteristics | मुख्य वैशिष्ट्ये

1. **Position**: Conjunctions typically appear between the elements they connect
2. **No inflection**: Unlike nouns and verbs, conjunctions don't change form
3. **Multiple functions**: Some conjunctions like 'तर' can mean 'if' or 'then' based on context

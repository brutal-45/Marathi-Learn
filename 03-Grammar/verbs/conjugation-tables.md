**English:** This reference page provides complete conjugation tables for the most common Marathi verbs across all tenses. Use this as a quick reference when practicing or when you need to look up a specific form.

**मराठी:** हे संदर्भ पृष्ठ सर्वात सामान्य मराठी क्रियापदांसाठी संपूर्ण रूप सारण्या प्रदान करते. सराव करताना किंवा विशिष्ट रूप शोधण्याची गरज असल्यास याचा द्रुत संदर्भ म्हणून वापर करा.

## 📚 Key Takeaways | मुख्य मुद्दे

1. **Present tense uses -त- with gender/number endings**
   **वर्तमान काळ -त- प्रत्यय वापरतो**

2. **Past tense uses -ल- with gender/number endings**
   **भूतकाळ -ल- प्रत्यय वापरतो**

3. **Future tense has unique endings for each person**
   **भविष्यकाळात प्रत्येक पुरुषासाठी वेगळे प्रत्यय आहेत**

4. **Some verbs have irregular roots in past tense**
   **काही क्रियापदांची भूतकाळातील मूळे अनियमित आहेत**

5. **Gender affects verb endings throughout**
   **लिंगाचा क्रियापद प्रत्ययांवर सर्वत्र परिणाम होतो**

# Adverbs Exercises | क्रियाविशेषण सराव

## Exercise 1: Identify the Adverb Type | क्रियाविशेषणाचा प्रकार ओळखा

Identify whether the underlined adverb is Time, Place, Manner, or Frequency.

| Sentence | Adverb Type |
|----------|-------------|
| 1. तो **नेहमी** वेळेवर येतो. | _______ |
| 2. **इथे** बसा. | _______ |
| 3. ती **हळू** बोलते. | _______ |
| 4. **आता** जा. | _______ |
| 5. पैसे **कुठेतरी** पडले आहेत. | _______ |
| 6. तो **कधीकधी** रागावतो. | _______ |
| 7. **बाहेर** चला. | _______ |
| 8. **लवकर** ये! | _______ |

<details>
<summary>🔍 Answers</summary>

1. Frequency (नेहमी = always)
2. Place (इथे = here)
3. Manner (हळू = slowly)
4. Time (आता = now)
5. Place (कुठेतरी = somewhere)
6. Frequency (कधीकधी = sometimes)
7. Place (बाहेर = outside)
8. Manner (लवकर = quickly)

</details>

## Exercise 3: Translate to Marathi | मराठीत अनुवाद करा

Translate the following sentences using appropriate adverbs:

1. Come here slowly.
   - _________________

2. I always speak the truth.
   - _________________

3. Don't go outside now.
   - _________________

4. She sometimes visits us.
   - _________________

5. Write neatly.
   - _________________

6. Where is my book?
   - _________________

7. He will come tomorrow.
   - _________________

8. Read carefully.
   - _________________

<details>
<summary>🔍 Answers</summary>

1. इथे हळू ये. / इथे हळू या. (respectively for informal/formal)
2. मी नेहमी सत्य बोलतो.
3. आता बाहेर जाऊ नको. / आता बाहेर जाऊ नका.
4. ती कधीकधी आपल्याला भेटते.
5. नीट लिही.
6. माझे पुस्तक कुठे आहे?
7. तो उद्या येईल.
8. काळजीपूर्वक वाच. / नीट वाच.

</details>

## Exercise 5: Sentence Construction | वाक्य रचना

Create sentences using the given adverbs:

1. **आता** (now)
   - _________________

2. **इथे** (here)
   - _________________

3. **नेहमी** (always)
   - _________________

4. **हळू** (slowly)
   - _________________

5. **बाहेर** (outside)
   - _________________

<details>
<summary>🔍 Sample Answers</summary>

1. आता घरी जा. (Go home now.)
2. इथे पुस्तक ठेव. (Keep the book here.)
3. तो नेहमी वेळेवर येतो. (He always comes on time.)
4. हळू बोल. (Speak slowly.)
5. बाहेर खेळायला जा. (Go outside to play.)

</details>

## Exercise 6: Choose the Correct Adverb | योग्य क्रियाविशेषण निवडा

Choose the correct adverb for each sentence:

1. तो _______ उशिरा येतो. (नेहमी / इथे)
2. पैसे _______ आहेत. (आता / इथे)
3. _______ वाच, परीक्षा आहे. (नीट / नेहमी)
4. आई _______ जेवण बनवते. (दररोज / कुठे)
5. _______ कुठे जात आहात? (हळू / कुठे)

<details>
<summary>🔍 Answers</summary>

1. नेहमी (always - makes sense for habitual action)
2. इथे (here - for location)
3. नीट (properly - for manner of reading)
4. दररोज (daily - for frequency)
5. कुठे (where - question about place)

</details>

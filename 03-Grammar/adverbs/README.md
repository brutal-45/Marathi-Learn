# Adverbs in Marathi | मराठी क्रियाविशेषण

## Introduction | परिचय

Adverbs (क्रियाविशेषण - *kriyavisheshan*) are words that modify verbs, adjectives, or other adverbs. They provide additional information about how, when, where, or to what extent an action occurs.

## Types of Adverbs | क्रियाविशेषणांचे प्रकार

| Type | Marathi Name | Description |
|------|--------------|-------------|
| Time | कालवाचक | When something happens |
| Place | स्थानवाचक | Where something happens |
| Manner | रीतिवाचक | How something happens |
| Frequency | वारंवारता | How often something happens |
| Degree | मात्रावाचक | To what extent |

## Key Characteristics | मुख्य वैशिष्ट्ये

### Position in Sentences
- Adverbs typically appear before the verb they modify
- Some adverbs can appear at the beginning of sentences for emphasis
- Unlike English, Marathi adverbs don't change form based on gender or number

### Formation
Many adverbs in Marathi are derived from:
- **Adjectives + ने/पणे**: चांगला (good) → चांगलेपणे (well)
- **Nouns + च**: दिवस (day) → दिवसच (daily)
- **Independent words**: आता (now), इथे (here)

## Common Adverbs | सामान्य क्रियाविशेषण

### Quick Reference Table

| English | Marathi | Transliteration | Example |
|---------|---------|-----------------|---------|
| now | आता | ātā | आता जा (Go now) |
| then | मग | mag | मग आलो (Then I came) |
| always | नेहमी | nehmi | नेहमी ये (Always come) |
| here | इथे | ithe | इथे बस (Sit here) |
| slowly | हळू | hāḷū | हळू चल (Walk slowly) |
| properly | नीट | nīṭ | नीट वाच (Read properly) |

# Learning Paths

Welcome to the Learning Paths section! This folder contains structured learning programs to help you learn Marathi systematically.

## 📚 Available Programs

### 🚀 [30-Day Challenge](./30-day-challenge.md)
A quick-start program for beginners who want to build a solid foundation in just one month. Perfect for motivated learners who can dedicate 30-60 minutes daily.

**Best for:** Complete beginners with limited time
**Time commitment:** 30-60 minutes/day
**Outcome:** Basic conversational ability

---

### 🎯 [90-Day Comprehensive Program](./90-day-program.md)
An in-depth learning journey covering all aspects of Marathi - reading, writing, speaking, and listening. Includes cultural context and advanced topics.

**Best for:** Serious learners aiming for intermediate proficiency
**Time commitment:** 1-2 hours/day
**Outcome:** Intermediate fluency and cultural understanding

---

### 📖 [Self-Study Guide](./self-study-guide.md)
A flexible, self-paced approach for independent learners. Customize your learning journey based on your goals, interests, and available time.

**Best for:** Self-directed learners with varying schedules
**Time commitment:** Flexible
**Outcome:** Personalized proficiency based on goals

---

### 📊 [Progress Tracker](./progress-tracker.md)
Templates and tools to track your learning journey. Includes daily, weekly, and monthly tracking sheets with milestone markers.

**Use with:** Any of the above programs

---

## 🗺️ How to Choose Your Path

| Your Situation | Recommended Path |
|----------------|------------------|
| Complete beginner, motivated | 30-Day Challenge |
| Want structured, thorough learning | 90-Day Program |
| Irregular schedule, prefer flexibility | Self-Study Guide |
| Already know basics, want to improve | Self-Study Guide (Intermediate track) |
| Preparing for travel to Maharashtra | 30-Day Challenge + Conversations focus |

## 📋 Prerequisites

Before starting any learning path, ensure you have:

- [ ] A dedicated notebook for practice
- [ ] Access to audio resources (headphones recommended)
- [ ] A practice partner or language exchange app (optional but helpful)
- [ ] 15-30 minutes of uninterrupted daily practice time

## 🎯 Learning Tips

1. **Consistency over intensity** - 20 minutes daily beats 2 hours once a week
2. **Speak from day one** - Don't wait until you're "ready"
3. **Immerse yourself** - Listen to Marathi music, watch movies
4. **Make mistakes** - They're essential for learning
5. **Review regularly** - Spaced repetition is key

## 🏆 Setting Goals

Use the SMART framework:
- **S**pecific - "Learn 50 new words" not "Learn vocabulary"
- **M**easurable - Track progress with numbers
- **A**chievable - Challenge yourself but stay realistic
- **R**elevant - Focus on what you'll actually use
- **T**ime-bound - Set deadlines and milestones

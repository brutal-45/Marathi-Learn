# 🌐 Useful Websites for Learning Marathi

A collection of websites to support your Marathi learning journey.

---

## Learning Platforms

### 1. **LearnMarathi.org**
- **URL**: https://www.learnmarathi.org
- **Level**: Beginner to Intermediate
- **Description**: Comprehensive website offering free lessons on Marathi alphabet, grammar, vocabulary, and conversation patterns.
- **Features**:
  - Structured lessons with audio
  - Grammar explanations
  - Practice exercises
  - Vocabulary lists
- **Recommendation**: ⭐⭐⭐⭐⭐ Best free resource for beginners

### 2. **iLanguages Marathi**
- **URL**: https://www.ilanguages.org/marathi.php
- **Level**: Beginner
- **Description**: Free language learning resource covering vocabulary, phrases, and grammar basics.
- **Features**:
  - Vocabulary flashcards
  - Basic phrases
  - Quiz feature
- **Recommendation**: ⭐⭐⭐⭐ Good for vocabulary building

### 3. **Memrise Marathi Courses**
- **URL**: https://www.memrise.com/courses/marathi/
- **Level**: All levels
- **Description**: Community-created courses using spaced repetition for vocabulary learning.
- **Features**:
  - Spaced repetition system
  - Audio pronunciation
  - Community courses
- **Recommendation**: ⭐⭐⭐⭐ Excellent for vocabulary retention

## Dictionary Websites

### 4. **MarathiDict**
- **URL**: https://www.marathidict.com
- **Level**: All levels
- **Description**: Online Marathi-English and English-Marathi dictionary with pronunciation guides.
- **Features**:
  - Bidirectional lookup
  - Example sentences
  - Synonyms and antonyms
- **Recommendation**: ⭐⭐⭐⭐⭐ Essential daily tool

### 5. **Shabdkosh Marathi**
- **URL**: https://www.shabdkosh.com/dictionary/english-marathi
- **Level**: All levels
- **Description**: Popular English-Marathi dictionary with detailed definitions and usage examples.
- **Features**:
  - Detailed definitions
  - Example sentences
  - Related words
  - Mobile app available
- **Recommendation**: ⭐⭐⭐⭐⭐ Highly recommended

### 6. **Wiktionary Marathi**
- **URL**: https://mr.wiktionary.org
- **Level**: All levels
- **Description**: Collaborative dictionary with extensive word entries in Marathi script.
- **Features**:
  - Open-source
  - Detailed etymology
  - Word forms and conjugations
- **Recommendation**: ⭐⭐⭐⭐ Good for advanced learners

## 📱 Tips for Using Websites Effectively

1. **Set a daily routine**: Visit at least one learning site daily
2. **Use multiple resources**: Different sites have different strengths
3. **Read news daily**: Even 5 minutes helps build vocabulary
4. **Listen while doing other tasks**: Passive listening improves familiarity
5. **Use browser extensions**: Add translation tools for quick lookup

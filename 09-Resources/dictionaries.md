## 💡 Tips for Using Dictionaries Effectively

### General Tips
1. **Don't translate word-by-word** - Understand context
2. **Note multiple meanings** - Words often have various uses
3. **Check example sentences** - Usage is as important as meaning
4. **Learn word families** - Related words help memory

### Look-up Strategy
1. **First attempt**: Try to understand from context
2. **Second attempt**: Quick online lookup (Shabdkosh)
3. **Deep study**: Molesworth for detailed understanding
4. **Practice**: Create sentences with new words

### Building Vocabulary
1. **Word of the day**: Many dictionaries offer this feature
2. **Bookmark unknown words**: Review them regularly
3. **Create flashcards**: Use Anki with dictionary entries
4. **Keep a vocabulary journal**: Write new words with examples

## 🔍 Understanding Dictionary Entries

### Marathi Word Entry Example:
```
देव (deva) - masculine noun
1. god, deity
2. (in compounds) divine, celestial
Example: देवाचे नाव घ्या (Say God's name / Begin a task)
Related: देवळ (temple), देवी (goddess)
```

### Key Abbreviations:
- **पु.** (पुल्लिंग) = Masculine
- **स्त्री.** (स्त्रीलिंग) = Feminine
- **न.** (नपुंसकलिंग) = Neuter
- **क्रि.** (क्रियापद) = Verb
- **वि.** (विशेषण) = Adjective

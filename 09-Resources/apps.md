# 📱 Mobile Apps for Marathi Learning

Curated list of mobile applications to help you learn Marathi on the go.

---

## Top Recommended Apps

### 1. **Duolingo**
- **Platform**: iOS, Android, Web
- **Price**: Free (Premium available)
- **Level**: Beginner
- **Description**: While Marathi is not yet available as a full course, users can practice related skills and stay updated for future releases.
- **Features**:
  - Gamified learning
  - Daily streaks
  - Bite-sized lessons
- **Recommendation**: ⭐⭐⭐ Check for future Marathi course
- **Link**: [App Store](https://apps.apple.com/placeholder) | [Play Store](https://play.google.com/placeholder)

---

### 2. **Memrise**
- **Platform**: iOS, Android, Web
- **Price**: Free (Premium available)
- **Level**: All levels
- **Description**: Offers community-created Marathi courses using spaced repetition and video clips of native speakers.
- **Features**:
  - Spaced repetition system
  - Native speaker videos
  - Offline mode (Premium)

- **Recommendation**: ⭐⭐⭐⭐⭐ Highly recommended for vocabulary
- **Link**: [App Store](https://apps.apple.com/placeholder) | [Play Store](https://play.google.com/placeholder)

## 💡 Tips for Using Apps Effectively

1. **Daily practice**: Even 10 minutes daily is better than 1 hour weekly
2. **Combine apps**: Use vocabulary apps + dictionary + speaking apps together
3. **Set reminders**: Enable daily notifications
4. **Track progress**: Use streak features to stay motivated
5. **Mix learning styles**: Use visual apps, audio apps, and writing apps

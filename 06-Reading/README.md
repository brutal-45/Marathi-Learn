Welcome to the Reading section! This section is designed to help you practice reading Marathi through progressively challenging materials.

## How to Use This Section

### Reading Strategy
1. **First Read**: Read the Devanagari text aloud, even if you don't understand everything
2. **Transliteration**: Use the transliteration to help with pronunciation
3. **Translation**: Read the English translation to understand meaning
4. **Vocabulary**: Study the vocabulary list for new words
5. **Second Read**: Re-read the Marathi text with better understanding
6. **Comprehension**: Answer the questions to test your understanding

### Tips for Effective Reading
- 📖 Read aloud to practice pronunciation
- ✍️ Write down new words in a notebook
- 🔄 Re-read passages multiple times
- 🎯 Focus on understanding context, not every word
- 📝 Try to answer comprehension questions without looking back

## Content Format

Each reading passage includes:
- **देवनागरी (Devanagari)**: Original Marathi text
- **Transliteration**: Romanized pronunciation guide
- **English Translation**: Complete English translation
- **Vocabulary List (शब्दसंग्रह)**: Key words with meanings
- **Comprehension Questions (प्रश्न)**: Questions to test understanding
